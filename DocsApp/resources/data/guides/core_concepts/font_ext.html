#Sencha Font Packages

Our new Triton theme needed a few additions that didn't exist in the base Font Awesome font. That said, we've created a small list of icons, codes, and names that you can utilize when including our custom font families.

We are also including the same information for the Pictos font family.

##Usage Details The Pictos icon font is not included in the framework by default (as was the case with Sencha Touch). It is now a separate package you’ll need to require in your application and / or theme.

// when requiring Pictos in a theme you’ll update the theme’s `package.json` file
// for applications you’ll need to update the app’s Sencha Cmd generated `app.json` file
“requires”: [
    “font-pictos”
]

Once required, you can use the following syntax for iconCls component configs in order to use icons from the Pictos library:

// pictos-{iconName} is used to set a named icon from the Pictos icon set
iconCls: 'pictos pictos-home'

A similar requirement step and iconCls syntax will be used if you prefer to use icons from the new “font-awesome” package.

“requires”: [
    “font-awesome”
    ]

// fa-{iconName} is used to set a named icon from the Font Awesome icon set
iconCls: ‘x-fa fa-home’
